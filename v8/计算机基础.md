## 语言

1. 机器指令 0101
2. 汇编语言 将0101指令转成单词 (难点:寻址方式)
3. 低级语言 C语言，面向对象的（难点:指针）
4. 高级语言 JAVA C++


### 汇编语言
1. 指令 + 数据，没有语句
2. 依赖硬件平台限制，可移植性很低
3. 直接操作硬件
4. 是最贴近底层的计算机语言
